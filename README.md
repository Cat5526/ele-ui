# vue-elementui
### 创建vue项目
vue create 项目名字，然后自己配置

## Project setup
```
npm install

## 还要安装这两个 在你项目名字文件夹里输入cmd，然后输入以下命令安装

npm install --save element-ui
npm install --save echarts
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
